## 🎯 项目目标

### 原始需求
实现一个"沉浸式的学习中心教学、实训与课研一体化设计"，采用"四区联动"的沉浸式教学界面设计，支持教学与课研模式的即时切换。

### 核心目标
- ✅ 实现四区联动布局（导航、教学、实践、AI 助教）
- ✅ 支持学习类和实践类任务的双层 Tab
- ✅ 实现深链接和状态持久化
- ✅ 实现前置依赖检查和进度追踪
- ✅ 对标参考设计进行 UI 优化

## 🔧 技术实现

### 核心技术栈
- **前端框架**: React 18 + TypeScript
- **UI 组件库**: shadcn/ui
- **路由**: React Router v6
- **状态管理**: React Hooks + localStorage
- **样式**: Tailwind CSS
- **测试**: Playwright + BDD

### 架构设计
- **可插拔 Tab 注册表** - 易于扩展新任务类型
- **深链接支持** - 支持分享和书签
- **状态持久化** - localStorage 保存用户偏好
- **iframe 隔离** - 安全可靠的外部服务集成
