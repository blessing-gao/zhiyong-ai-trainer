## 📊 错题集统计卡片

显示四个关键指标：

| 指标 | 说明 | 示例 |
|------|------|------|
| 错题总数 | 用户收集的所有错题数量 | 5 道 |
| 平均正确率 | 所有错题的平均正确率 | 65% |
| 最后复习 | 最后一次复习错题的时间 | 3 天前 |
| 已掌握 | 已经掌握的错题数量 | 2 道 |

---

## 📝 错题列表功能

### 错题信息展示

每道错题显示以下信息：

1. **题目基本信息**
   - 题号（第 1 题、第 2 题等）
   - 题型（单选题、多选题、判断题）
   - 所属分类（深度学习、计算机视觉等）
   - 难度等级（简单、中等、困难）

2. **错题统计**
   - 错误次数（用户做错的次数）
   - 正确率（用户答对的概率）

3. **答题对比**
   - 用户答案（红色显示）
   - 正确答案（绿色显示）
   - 详细解析（帮助理解）

4. **操作按钮**
   - "重新练习" - 重新做这道题

## 📋 错题数据结构

```typescript
interface WrongQuestion {
  id: number;
  question: string;           // 题目内容
  type: string;              // 题型（单选题、多选题、判断题）
  category: string;          // 分类（深度学习、计算机视觉等）
  difficulty: string;        // 难度（简单、中等、困难）
  wrongCount: number;        // 错误次数
  lastWrongDate: string;     // 最后错误日期
  correctRate: string;       // 正确率
  options: string[];         // 选项列表
  correctAnswer: string;     // 正确答案
  yourAnswer: string;        // 用户答案
  explanation: string;       // 详细解析
}
```

## 💡 使用说明

### 访问错题集

1. 登录系统
2. 进入个人中心
3. 点击"错题集"标签页
4. 查看错题统计和详细列表

### 复习错题

1. 在错题列表中找到要复习的题目
2. 点击"重新练习"按钮
3. 重新做题并对比答案
4. 查看详细解析

### 导出错题集

1. 点击"导出错题集"按钮
2. 选择导出格式（PDF/Excel）
3. 下载到本地
